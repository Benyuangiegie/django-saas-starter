Django REST API Test Page

Welcome to your Django REST Framework API! Below are the available endpoints:

GET /api/docs/

Swagger UI documentation for all API endpoints

GET /api/schema/

OpenAPI schema in JSON format

POST /api/v1/auth/users/

Create a new user account

{ "email": "user@example.com", "password": "yourpassword", "password_confirm": "yourpassword", "first_name": "John", "last_name": "Doe" }

POST /api/v1/auth/login/

Login and get access tokens

{ "email": "user@example.com", "password": "yourpassword" }

GET /api/v1/auth/users/me/

Get current user profile (requires authentication)

GET /api/v1/tenants/

List all tenants (requires authentication)

POST /api/v1/tenants/

Create a new tenant (admin only)

{ "name": "Acme Corporation", "slug": "acme-corp", "contact_email": "admin@acme.com", "max_users": 50 }

GET /api/v1/tenants/{id}/

Get tenant details

POST /api/v1/tenants/{id}/activate/

Activate a tenant (admin only)

GET /api/v1/tenants/{id}/stats/

Get tenant statistics

POST /api/v1/auth/token/refresh/

Refresh access token

{ "refresh": "your_refresh_token" }

POST /api/v1/auth/logout/

Logout and blacklist refresh token

{ "refresh": "your_refresh_token" }

Quick Start

  1. Create a user account using POST /api/v1/auth/users/
  2. Login using POST /api/v1/auth/login/ to get access tokens
  3. Include the access token in the Authorization header: Bearer <access_token>
  4. Access protected endpoints like GET /api/v1/auth/users/me/
  5. Visit /api/docs/ for interactive API documentation
